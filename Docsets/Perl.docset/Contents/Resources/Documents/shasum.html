NAME

shasum - Print or Check SHA Checksums

SYNOPSIS

  1.  Usage: shasum [OPTION]... [FILE]...
  2.  Print or check SHA checksums.
  3.  With no FILE, or when FILE is -, read standard input.
  5.    -a, --algorithm   1 (default), 224, 256, 384, 512, 512224, 512256
  6.    -b, --binary      read in binary mode
  7.    -c, --check       read SHA sums from the FILEs and check them
  8.    -p, --portable    read files in portable mode
  9.                          produces same digest on Windows/Unix/Mac
  10.    -t, --text        read in text mode (default)
  12.  The following two options are useful only when verifying checksums:
  13.    -s, --status      don't output anything, status code shows success
  14.    -w, --warn        warn about improperly formatted checksum lines
  16.    -h, --help        display this help and exit
  17.    -v, --version     output version information and exit
  19.  When verifying SHA-512/224 or SHA-512/256 checksums, indicate the
  20.  algorithm explicitly using the -a option, e.g.
  22.    shasum -a 512224 -c checksumfile
  24.  The sums are computed as described in FIPS-180-4.  When checking, the
  25.  input should be a former output of this program.  The default mode is to
  26.  print a line with checksum, a character indicating type (`*' for binary,
  27.   ` ' for text, `?' for portable), and name for each FILE.
  29.  Report shasum bugs to mshelor@cpan.org

DESCRIPTION

Running shasum is often the quickest way to compute SHA message digests. The user simply feeds data to the script through files or standard input, and then collects the results from standard output.

The following command shows how easy it is to compute digests for typical inputs such as the NIST test vector "abc":

  1. 	perl -e "print qq(abc)" | shasum

Or, if you want to use SHA-256 instead of the default SHA-1, simply say:

  1. 	perl -e "print qq(abc)" | shasum -a 256

Since shasum mimics the behavior of the combined GNU sha1sum, sha224sum, sha256sum, sha384sum, and sha512sum programs, you can install this script as a convenient drop-in replacement.

SEE ALSO

shasum is implemented using the Perl module Digest::SHA or Digest::SHA::PurePerl.
